Descripción de la aplicación:

Para este proyecto hemos decidido realizar una página web dedicada a la creación de un listado de recetas personales. La aplicación web consiste en una base de datos con los nombres de los platos, sus ingredientes y los pasos para su realización. También se podrá incluir una imágen del mismo.

Los usuarios registrados podrán acceder a la lista de recetas para añadir o modificar una receta edntro de la base de datos, mientras que los anónimos sólo podrán ver las recetas.

Para completar la aplicación, se incluirá un buscador de recetas por nombre.
